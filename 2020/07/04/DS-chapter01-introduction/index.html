数据结构 第一章 绪论

（2）算法的定义、算法的基本特性以及算法分析的基本概念。

错题

  • P3：1、3、4
  • P4：7、1
  • P7：2、6
  • P8：12

1.1 数据结构的基本概念

数据：数据是信息的载体，用于描述客观事物。

数据元素：数据的基本单位，一个数据元素包含多个数据项。例如，学生记录就是一个数据元素，它包含学号、姓名等数据项。

数据对象：具有相同性质的数据元素的集合。

抽象数据类型：

  • 定于：数学模型和定义在该模型上的一组操作
  • 组成要素：
    • 数学模型：指明了数据对象和数据关系；即数据的逻辑结构和存储结构
    • 基本操作集：定义的操作即如何实现数据的逻辑结构

数据结构：

  • 定义：数据元素彼此不是孤立的，他们之间的关系称为结构。数据结构是相互之间存在关系的数据元素的集合。
  • 三要素：
    • 逻辑结构：线性结构、非线性结构
    • 存储结构：顺序存储、链式存储、索引存储、散列存储（Hash存储）
    • 定义的操作：如何实现数据的逻辑结果

1.2 算法

定义：对特定问题求解步骤的一种描述

基本特性：

  • 有穷性
  • 确定性：算法每一步的意思必须明确
  • 可行性：算法中的每一步都必须是可行的
  • 输入
  • 输出

算法分析：

  • 时间复杂度T(n)
  • 空间复杂度S(n)
