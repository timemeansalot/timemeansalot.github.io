基础30讲之高等数学预备知识

1.1 函数的概念与特性

  1. 有界性：有界性必须指明在哪个区间有界

  2. 奇偶性：祖孙三代奇偶性交替变化

    $F(x)=f(x)-f(-x)必为奇函数;F(x)=f(x)+f(-x)必为偶函数$

  3. 周期性：下式表明f以T为周期：$f(x)=f(2T-x)或者f(T-x)=f(T+x)$

1.2 函数的图像

1.2.1 三角函数图像

  1. 正切函数$y=tanx$

  2. 余切函数$y=cotx$

  3. 反三角函数$y=arcsinx$

  4. 反余弦函数$y=arccosx$

  5. 反正切函数$y=arctanx$

  6. 反余切函数$y=arccotx$

1.2.2 极坐标系图像

  1. 心形线$r=a(1-cos\theta)$

  2. 玫瑰线$r=asin3\theta (a>0)$

  3. 阿基米德螺线$r=a\theta(a>0,\theta\geqslant0)$

  4. 伯努力双纽线$r^2=a^2cos2\theta或者r^2=a^2sin2\theta$
