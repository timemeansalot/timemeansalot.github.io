Tensorflow从源码编译安装tensorflow-gpu版本

教你安装NVIDIA驱动，安装cuda，cudnn，bazel

教你通过使用bazel编译tensorflow源文件安装tensorflow-gpu版本

[![VouAAI.md.png](https://s2.ax1x.com/2019/06/15/VouAAI.md.png)](https://imgchr.com/i/VouAAI)

tensorflow

tensorflow-gpu的安装主要分为4步：

  • 更新nvidia驱动
  • 安装Cuda和cuDNN
  • 安装Bazel
  • 编译安装tensorflow-gpu版本

Step1：更新驱动

更新驱动有两种方式，一种是在终端通过网络更新，一种是手动更新

方法一：手动更新

首先我们要下载418版本的NVIDIA驱动

1. 禁止Nouveau驱动，避免开机卡在logo处

2. 卸载低版本的nvidia驱动

3. 关闭lightdm

重启电脑，按ctrl+alt+F2进入x-server
输入用户名和密码

4. 执行安装程序

5. 重启lightdm

至此，驱动更新完成，可在终端验证

方法二：网络安装

前两步和手动安装相同

5.安装驱动

至此，驱动更新完成，可在终端验证

Step2 ：安装CUDA和cuDNN

CUDA是NVIDIA开发的用于科学技术的工具，cuDNN是配合CUDA使用的。

在安装他们之前，我们首先要根据我们需要的tensorflow版本来确定我们需要的cuda和cuDNN版本,不然就会出错。因为我们使用tensorflow1.13，所以我们需要Cuda10.0和cuDNN7.6.0

一：安装CUDA10.0

1.下载

去nvidia官网地址下载10.0版本的cuda,下载.sh文件。如图所示

２．安装

不要选择为显卡安装加速驱动，因为我们已经在第一步时安装了驱动了

1.下载cuDNN
